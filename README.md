# ClientSphere CRM Dashboard
A modern Lead Management CRM Dashboard built using HTML, CSS, and JavaScript.
## Features
- Add new clients/leads
- Delete client records
- Lead status tracking
- Dashboard statistics cards
- Responsive modern UI
- Interactive animations
- Local storage support
## Technologies Used
- HTML
- CSS
- JavaScript
## Project Purpose
This project was created as Task 2 for the Future Interns Full Stack Web Development Internship Program to improve frontend development and JavaScript DOM manipulation skills.
